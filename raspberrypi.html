Building a Web Server.

Using a Raspberry Pi to build a web server was easier than I expected. I'm using a standard Raspbian "wheezy" image, and then installed SSH, Apache, and PHP. I have a static IP address with my ISP, so that's good. I just needed to point my router at my Pi, to let the world see it. Then I created a website, and I was up and running.

I can't remember which websites I used for detailed instructions, but a search will bring up quite a few.

I used Apache on my server, but a number of blogs suggest that Nginx might be a better one for the Pi, as it's designed to be more light-weight than Apache, and should run a bit faster. So I might give it a try sometime. But for the time being, my current Pi Server with Apache, seems to be running surprising well.

Our Pi Case.

My son wanted a case for his Pi, so we decided to make one ourselves. We cut a couple of rectangular pieces of clear acrylic, and assembled them with spacers, screws and rubber grommets.

I think it turned out quite well.
