Discovery 3 Photo Gallery

I've created this little photo gallery of my Discovery 3. The idea was to see how a number of photos could be loaded into a browser, from the Pi Server. I think the main limitation is the upload speed of my ADSL connection, which is usually around 0.8Mbps.

So I decided to create small .jpg files (around 50kb each) to make the gallery. Then, clicking on a small photo will link the user to a larger file which is stored and downloaded from Flickr. This means the user doesn't need to view the large file via my slow ADSL connection. I think this works quite well.
